# 仕様
インスタグラムには自動投稿APIが無いため、スクレイピングを利用した自動投稿ツールです。
tkinterで作成をしています。

# インターバル実行画面
1分/10分/30分/60分毎と、指定した時間間隔で定期的に投稿します。
待ち時間は進捗バーにて可視化しています。
